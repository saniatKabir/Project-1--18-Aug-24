Task-1: Linking Pages

External Link

Linking a external page from the web can be done by adding hyperlink of that perticular page. For example: we can go to the page of ssb641 by clicking on Facebook. Presently, if anyone clicks on the word "Facebook" it will open in a new tab since target is set to "_blank". If "_blank" was not set as target, it would open on the same tab. It is to be noted that if the hyperlink is not available insert # in the hyperlink value

Internal Link

In case of internal link, there are mainly two kinds of link namely forward link and backward link.

Forward Link

During Forword Link, there may be mainly three situations: index.htlm file and the file or any object to be linked may be in the same directory, root directory, or in different directory like sub-folder or sub-sub.... folder. When other folders are linked in index.html file or the 'HOME' page, normally those are linked following forword link procedure.

In same directory: If the index.htlm file is on the same directory it can be linked by inserting the name in the valu of hyperlink:

HOME | PAGE-1

If the files are in deffent directory: In this case the file will be linked with the forward slash following forward linking method. In this case use "folder name/filename.html" to be linked. For example:

HOME | PAGE-1 PAGE-2 PAGE-3

Backward Link

For back ward link you manytime do you need to go back from present file location. One must use that number of "../" before adding the folder name/file name
